# Sleep Timer

An Android sleep timer app with device lock capability. Set a timer and your device will
automatically lock when time runs out â€” perfect for falling asleep to music or podcasts.

## Features

- Configurable sleep timer with customizable duration
- Device lock when timer expires
- Media fade-out before lock
- Localization support (English, Russian)
- Edge-to-edge UI
- Predictive back gesture support

## Tech Stack

- **Language**: Kotlin
- **UI**: Jetpack Compose with Material 3
- **Architecture**: Clean Architecture with Decompose navigation
- **DI**: Metro
- **Persistence**: DataStore
- **State Management**: Kotlin Flow / StateFlow
- **Min SDK**: 24 (Android 7.0)
- **Target SDK**: 36
